USAGE:
You schould use linux Debian or similar distributions (such as Ubuntu, Mint etc).

1) Clone the repo
2) Cd the repo
3) Make the command "make" or "make all"
4) The library MLX42 schould be cloned and you will see the new dir with the name MLX42
5) You also have a lot of maps in the dir "maps/good"
6) Try to run the program with such parameters: ./cub3d maps/good/test.cub
7) The window schould open and you will see the game world
